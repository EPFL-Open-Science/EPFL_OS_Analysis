# EPFL_OS_Analysis

Early state of development. 

## How to run this project

Obtain Token from Github and Zenodo and place them in a `.env` file. 

```
GITHUB_TOKEN=Token <GITHUB TOKEN>
ZENODO_TOKEN=<ZENODO TOKEN>
```
